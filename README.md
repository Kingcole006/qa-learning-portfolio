## 🚀 Running Tests

### Prerequisites
```bash
npm install
```

### Execute Tests
```bash
# Quick test run
npm test

# With HTML report
npm run test:report

# Verbose output
npm run test:verbose

# Clean old reports and run
npm run test:clean
```

### Manual Newman Execution
```bash
newman run data-driven-testing.json -e qa-env.json -d test-users.csv -r htmlextra --reporter-htmlextra-export ./reports/test-report.html
```

### Playwright Tests
```bash
# Navigate to Playwright folder
cd JavaScript-Fundamentals/Day16-Playwright

# Run basic test
node 01-basics.js

# Run QA Wolf solution
node 08-qawolf-solution.js
```
